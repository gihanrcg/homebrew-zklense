# Homebrew Tap for zklense

This is a Homebrew tap for [zklense](https://github.com/jinali98/zk-profiling-solana), a command-line tool for profiling, building, and deploying zero-knowledge proofs built with Noir for Solana Blockchain.

## Installation

```bash
brew tap gihanrcg/zklense
brew install zklense
```

## Update

```bash
brew update
brew upgrade zklense
```

## Uninstall

```bash
brew uninstall zklense
brew untap gihanrcg/zklense
```

## About zklense

zklense is a CLI tool designed for working with zero-knowledge proofs on the Solana blockchain using the Noir programming language.
